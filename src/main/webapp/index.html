Welcome to EDA Vessel Course Simulator

This app simulate vessels travel around routes in north pacifica. It is built from the Appsody project, which is an open source project that provides a set of open source tools and capabilities that simplifies the process of building and deploying cloud native applications. The MicroProfile project is an open community with the aim of optimizing Enterprise Java for a microservices architecture. Running on Open Liberty: an open community supporting Java EE and MicroProfile, a runtime starting fast and made for running cloud-native microservices.
