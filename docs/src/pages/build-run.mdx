## Pre-Requisites

* [Maven](https://maven.apache.org/install.html) used to compile and package the application.
* Java 8 or higher: Any compliant JVM should work.
  * [At least Java 8 JDK from Oracle](http://www.oracle.com/technetwork/java/javase/downloads/index.html)
* Have docker engine installed on your computer.
* Docker compose if you want to run kafka and the code locally.
